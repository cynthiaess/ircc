Effectuez une évaluation des compétences linguistiques

Vous devez prouver vos compétences linguistiques en passant un examen des compétences linguistiques approuvé.

Pour ce faire, vous devez :

  • prendre un rendez-vous pour faire évaluer vos compétences par un organisme approuvé par IRCC et payer le coût du test
  • entrer les résultats du test dans votre profil d’Entrée express (et le formulaire ou numéro de certificat des résultats de l'examen de compétences linguistiques et NIP de l'examen, si le système le demande)
Si un handicap vous empêche d’effectuer un ou plusieurs volets de l’examen des compétences linguistiques

Si vous souffrez d’un handicap et que vous n’êtes pas en mesure de remplir une ou plusieurs sections de votre évaluation des compétences linguistiques :

  • Vous devez utiliser notre outil de calcul des résultats moyens pour les examens des compétences linguistiques pour déterminer vos résultats moyens pour les compétences que vous n’avez pas pu remplir (si l’organisme d’évaluation des compétences linguistiques ne vous a pas donné directement vos résultats moyens).
  • Utilisez ces résultats moyens dans votre profil, dans le créateur de profil d’Entrée express.
  • Nous vérifierons à la fois les résultats moyens et les résultats des examens linguistiques que vous aurez soumis.

Examens des compétences linguistiques que nous acceptons

Vous pouvez passer n’importe lequel de ces examens approuvés :

Français Anglais
TEF Canada: Test d’évaluation de français

CELPIP: Canadian English Language Proficiency Index Program (en anglais seulement)

  • Vous devez passer l’examen CELPIP-General.

TCF Canada : Test de connaissance du français

IELTS : International English Language Testing System (en anglais seulement)

  • Vous devez choisir l’option General Training.
  • Nous n’acceptons pas les résultats du programme « One Skill Retake » du IELTS dans le cadre d’Entrée express.

PTE Core: Pearson Test of English (en anglais seulement)

  • Vous devez choisir l’option PTE Core.

Exigences linguistiques

Pour mesurer vos compétences en français ou en anglais, nous utilisons :

  • Les Niveaux de compétence linguistique canadien (NCLC), pour le français;
  • Les Canadian Language Benchmarks (CLB), pour l’anglais.

Catégorie de l’expérience canadienne

Les compétences dont vous aurez besoin dépendront du groupe dans lequel l’emploi est classé selon le système de la Classification nationale des professions (CNP).

Durée de validité de vos résultats

Les résultats de votre test ne doivent pas remonter à plus de 2 ans lorsque vous :

  • remplissez votre profil d’Entrée express, et
  • présentez votre demande de résidence permanente.
Si vous recevez une invitation à présenter une demande

Vérifiez si les résultats de votre test de compétence linguistique seront toujours valides au moment où vous envisagez de présenter votre demande.

Si vous n’avez pas reçu d’invitation à présenter une demande

Assurez-vous que vos résultats seront valides pour le reste de la période durant laquelle vous serez inscrit dans le bassin.

Si les résultats de votre test arrivent à expiration avant que vous ne puissiez présenter une demande, vous devriez alors :

  • soit passer le test de nouveau;
  • soit présenter votre demande avant l’expiration de vos résultats;
  • soit décliner l’invitation et retourner dans le bassin afin d’être sélectionné à une date ultérieure.

Si vous présentez une demande de résidence permanente alors que les résultats de votre test de compétence linguistique ne sont plus valides, nous refuserons votre demande.
