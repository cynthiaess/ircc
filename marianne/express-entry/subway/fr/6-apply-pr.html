1. Remplir le formulaire de demande

Si nous vous invitons à présenter une demande de résidence permanente, vous recevrez un message dans votre compte vous informant :

  • du programme dans le cadre duquel vous êtes invité;
  • des prochaines étapes.

Votre invitation à présenter une demande est valide pendant 60 jours seulement. Vous devriez commencer à remplir le formulaire tout de suite afin que vous puissiez obtenir tous les renseignements et les documents dont vous avez besoin avant l’expiration de votre invitation.

Pour remplir le formulaire :

  1. Ouvrez une session dans votre compte.
  2. Allez à la section « Continuer à remplir ma demande pas encore présentée ».
  3. À la droite de la rangée où le « type de demande » est « résidence permanente », sélectionnez le bouton bleu « Continuer la demande ».

Ouvrir une session dans votre compte sécurisé d’IRCC

Si vous avez déjà fait une demande d’IRCC

Le formulaire vous demandera si vous avez fait une demande d’IRCC dans le passé. Cela veut dire une demande complète, par exemple pour un visa, un permis de travail ou un autre programme d’immigration. Remplir un profil d’Entrée express ne compte pas.

Il vous demandera aussi si vous avez déjà fait une demande d’Entrée express. Ici aussi, vous devriez seulement répondre « oui » si vous avez rempli une demande de résidence permanente que vous avez soumise électroniquement. L’existence d’un profil, même si vous avez reçu une invitation à présenter une demande, ne compte pas comme une demande en soit.

Si vous n’êtes pas invité(e) à présenter une demande

Si vous n’êtes pas invité(e) à présenter une demande de résidence permanente dans les 12 mois suivant la soumission de votre profil d’Entrée express, celui-ci expirera automatiquement. Vous pouvez voir la date d’expiration de votre profil dans votre compte.

Lorsque votre profil expire, il est retiré du système. Vous recevrez un message dans votre compte.

Vous pouvez créer un nouveau profil. Si votre profil est sur le point d’expirer, vous pouvez faire des saisies d’écran pour qu’il soit plus facile de le remplir à nouveau.

Ne créez pas de nouveau profil avant l’expiration de votre profil existant. Si vous le faites, vous devez d’abord retirer votre profil existant.

Pour retirer votre profil :

  1. Connectez-vous à votre compte.
  2. Rendez-vous dans la section « Voir les demandes soumises » sur la page principale.
  3. Cliquez sur « Retrait du profil ».
  4. Cliquez sur le bouton « Continuer » sous la rubrique « Que voulez-vous faire aujourd’hui? », en dessous du texte « Avertissement! ».

Si vous ne soumettez pas de nouveau profil, vous ne pourrez pas mentionner Entrée express dans vos profils sur les sites d’affichage d’emplois.

Si vous en soumettez un, vous obtiendrez :

  • un nouveau numéro de profil d’Entrée express;
  • un nouveau code de validation de chercheur d’emploi pour le Guichet-Emplois.

Autres moyens d’immigrer

Vous pouvez envisager de présenter une demande directement à une province ou à un territoire dans le cadre du Programme des candidats des provinces (PCP) ou de trouver d’autres façons d’immigrer au Canada.

Si vous refusez une invitation à présenter une demande

Si vous déclinez une invitation, votre candidature sera remise dans le bassin d’Entrée express. Elle sera prise en compte lors de prochaines rondes d’invitations, si vous êtes toujours admissible.

La note de passage pour la sélection des candidats varie d’une ronde à l’autre. Si votre note n’est pas assez élevée, il se pourrait que vous ne receviez pas d’autre invitation à présenter une demande. Dans certains cas, il se pourrait que vous ayez à prendre des mesures afin d’être à nouveau invité à présenter une demande. Vous pourriez avoir, par exemple :

  • à améliorer votre note;
  • à mettre à jour votre profil d’Entrée express.

Si vous décidez de revenir dans le bassin, le fait que vous ayez décliné une invitation n’aura pas d’incidence sur vos chances de recevoir une nouvelle invitation plus tard. Toutefois, rien ne garantit que vous serez à nouveau invité à présenter une demande.

Si vous recevez une invitation, que vous ne la déclinez pas, mais que vous ne présentez pas de demande dans les 60 jours, l’invitation expirera et votre candidature sera retirée du bassin. Pour que votre candidature puisse être sélectionnée lors d’une prochaine ronde d’invitations, vous devrez :

  • remplir et soumettre un nouveau profil d’Entrée express;
  • satisfaire aux critères d’admissibilité;
  • être de nouveau accepté dans le bassin.

Si votre situation personnelle change

Si votre situation (ou celle de votre époux ou conjoint de fait) change, vous devriez recalculer votre note avant de présenter une demande de résidence permanente en ligne.

Si votre note recalculée est inférieure à celle du candidat de la ronde d’invitations le moins bien classé, vous devriez décliner l’invitation. Si vous décidez de présenter une demande quand même, nous pourrions :

  • refuser votre demande;
  • ne pas vous rembourser les frais relatifs à votre demande.

Voici des exemples de changements qui pourraient faire baisser votre note :

  • vous n’avez plus d’offre d’emploi valide;
  • vous ne figurez plus parmi les candidats désignés d’une province;
  • votre résultat à l’évaluation des compétences linguistiques :
    • a expiré;
    • est inférieur à ce qu’il était à la suite d’une nouvelle évaluation.

Si votre permis de travail expire avant que vous ne présentiez une demande

Si votre permis de travail expire avant que vous ne présentiez une demande, votre note pourrait baisser. Par exemple, cela peut se produire si vous avez obtenu points pour une offre d’emploi en fonction du permis de travail expiré.

Si votre note se trouve sous le seuil minimal :

  • Vous devriez refuser l’invitation.
  • Si vous soumettez tout de même votre demande, nous la refuserons.

Après que vous ayez refusé l’invitation :

  • Vous serez réintégré(e) au bassin de candidats.
  • Vous devrez mettre à jour votre profil Entrée express afin de modifier votre statut d’emploi.

2. Téléverser vos documents

Rassembler les documents requis

Vous devrez téléverser des copies des documents que vous avez utilisés pour créer votre profil. Le système vous offrira une liste de contrôle personnalisée lorsque vous aurez rempli le formulaire en ligne.

La plupart des demandeurs auront aussi besoin de téléverser ce qui suit :

  • des certificats de police;
  • une preuve de fonds suffisants;
  • un certificat de naissance, si vous déclarez avoir des enfants à votre charge;
  • le formulaire Recours aux services d’un représentant (PDF, 215 Ko), si vous retenez les services d’un représentant;
  • le formulaire Déclaration officielle d’union de fait (PDF, 2,30 Mo), si vous avez déclaré que votre état matrimonial est « en union de fait »;
  • un certificat de mariage, si vous avez déclaré que votre état matrimonial est « marié » ou «  mariée »;
  • un certificat de divorce et une entente de séparation légale, si vous avez déclaré que votre état matrimonial est « divorcé » ou « divorcée »;
  • un certificat de décès, si vous avez déclaré que votre état matrimonial est « veuf » ou « veuve »;
  • un certificat d’adoption, lorsqu’il est indiqué qu’un enfant à votre charge est « adopté ».
Autres documents pouvant être nécessaires

Vous avez seulement besoin de soumettre les documents ci-dessous si nous les avons inclus dans votre liste de contrôle personnalisée des documents :

  • preuve d’un lien avec un parent au Canada;
  • photographies numériques pour confirmer votre identité;
  • autre nom pour confirmer les alias;
  • autorisation de communiquer des renseignements personnels à une personne désignée (PDF, 1,83 Mo);
  • tout autre document qui, selon vous, est pertinent dans le cadre de votre demande.

Votre demande pourrait être refusée ou rejetée si vous ne soumettez pas les documents ci-dessus. Ces derniers sont utilisés afin de vérifier si vous satisfaites aux exigences du programme au titre duquel vous présentez une demande.

Traduction de documents

Si l’un de vos documents est rédigé dans une langue autre que le français ou l’anglais, vous devez également téléverser :

  • une traduction en français ou en anglais qui :
    • porte le timbre d’un traducteur agréé
      • Le traducteur doit photocopier votre document original et l’estampiller s’il s’agit du texte de départ pour sa traduction.
      • Les timbres et les sceaux qui ne sont ni en anglais ni en français doivent également être traduits. Vous pouvez joindre une copie numérisée d’une lettre indiquant que vous n’avez pas besoin d’un affidavit, car vous avez retenu les services d’un traducteur agréé.
    • ou
    • est accompagnée d’un affidavit de la personne qui a traduit le document
      • N’utilisez cette option que si le document ne peut pas être traduit par un traducteur agréé.
      • Le document traduit et le document original utilisé par le traducteur pour faire sa traduction doivent être mentionnés dans l’affidavit.
  • et
  • une copie numérisée du document original ou une copie numérisée d’une photocopie certifiée du document de départ original utilisés par le traducteur;
    • Vous devez seulement fournir une copie numérisée de la photocopie certifiée du document original si le document de la photocopie certifiée a servi comme texte de départ pour la traduction.

Traducteur agréé

Un traducteur agréé est membre en règle d’une association professionnelle de traducteur au Canada ou ailleurs. Son statut de traducteur agréé doit être confirmé par un sceau ou un timbre dans lequel figure son numéro de membre.

Tous les sceaux et timbres dans une langue autre que le français ou l’anglais doivent également être traduits.

Tout membre de la famille, représentant ou consultant du demandeur qu’il soit avocat, notaire ou traducteur n’est pas autorisé à traduire des documents.

Si le traducteur est en cours d’obtention de sa certification ou de son accréditation, il n’est pas considéré comme traducteur agréé aux fins d’IRCC.

Photocopie certifiée

Qu'est-ce qu'une photocopie certifiée

Une photocopie certifiée est une photocopie d’un document original. Elle doit être lisible et certifiée en tant que copie authentique de l’original par une personne autorisée.

La personne compare les documents et inscrit sur la photocopie :

  • son nom et y appose sa signature;
  • son poste ou titre;
  • le nom du document original;
  • la date où elle a certifié le document;
  • la phrase « J’atteste qu’il s’agit d’une copie certifiée conforme à l’original ».

Who can certify photocopies

Au Canada, les personnes autorisées qui peuvent certifier les copies de vos documents originaux comprennent les suivantes :

  • notaire public;
  • commissaire à l’assermentation;
  • commissaire aux affidavits.

Vérifiez auprès des autorités provinciales ou territoriales pour savoir quelles personnes sont autorisées à certifier vos documents.

À l’extérieur du Canada, chaque pays dispose d’autorités différentes qui certifient des documents. Il est possible qu’un notaire public puisse certifier vos documents, mais vous devriez vérifier si c’est le cas auprès des autorités locales pour en être certain.

Les membres de votre famille et vous-même ne pouvez pas certifier des copies de vos documents. Dans ce cas, un membre de la famille désigne :

  • un parent;
  • un tuteur;
  • un frère ou une sœur;
  • un époux;
  • un conjoint de fait;
  • un partenaire conjugal;
  • un grand-parent;
  • un enfant;
  • une tante;
  • un oncle;
  • une nièce;
  • un neveu;
  • un cousin.

Affidavit

Qu’est-ce qu’un affidavit

Un affidavit pour une traduction est un document qui assure que la traduction est fidèle au texte original.

Le traducteur jure, devant un commissaire autorisé à faire prêter serment dans le pays où il réside, que sa traduction est une version exacte du document original. Le commissaire ou le notaire public doit maîtriser l’anglais ou le français pour pouvoir faire prêter le serment.

Qui peut recevoir un affidavit

Au Canada, un affidavit peut être reçu par un :

  • notaire public;
  • commissaire aux serments;
  • commissaire aux affidavits.

À l’extérieur du Canada, un affidavit peut être reçu par un notaire public ou l’équivalent.

Qui ne peut pas recevoir d’affidavit

Les affidavits ne doivent pas être pris par les personnes suivantes :

  • les demandeurs eux-mêmes;
  • les membres de la famille du demandeur, même s’ils sont avocats, notaires ou traducteurs agréés, y compris :
    • parents,
    • tuteurs,
    • frères et sœurs,
    • époux,
    • conjoints de fait,
    • partenaires conjugaux,
    • grands-parents,
    • enfants,
    • oncles et tantes,
    • nièces et neveux,
    • cousins germains.
Comment signer des formulaires PDF

Signature manuscrite : imprimez le formulaire et signez-le à la main.

Signature numérique : insérez cette signature sur le formulaire PDF. Vous pouvez taper votre nom, utiliser DocuSign ou utiliser une signature sécurisée.

Numérisation et téléversement

Rassemblez tous les documents qui figurent sur votre liste de contrôle et faites des copies électroniques à l’aide d’un scanneur ou d’une caméra. Assurez-vous d’utiliser un des formats acceptés..

Une fois que vos documents sont prêts, téléversez-les dans votre liste de contrôle des documents.

Subir un examen médical, au besoin

Un examen médical aux fins de l’immigration (EMI) auprès d’un médecin désigné est requis dans le cadre d’une demande de résidence permanente. Les membres de votre famille doivent également passer un examen, même s’ils ne vous accompagnent pas.

Si vous n’avez jamais subi un examen médical
  1. Téléversez un document vierge dans le champ de téléversement de l’outil de création de profils d’Entrée express lorsque vous soumettez votre demande.
  2. Attendez que nous vous envoyions des instructions concernant la date à laquelle vous devrez subir un examen médical.

Nous n’approuverons pas votre demande de résidence permanente si votre état de santé :

  • présente un risque pour la santé ou la sécurité publique du Canada;
  • représenterait un fardeau excessif pour les services sociaux ou de santé du Canada.
Si vous avez déjà subi un examen médical

Si vous avez déjà subi un EMI dans le cadre d’une demande antérieure (par exemple, pour travailler ou étudier au pays) et que vous vivez actuellement au Canada, vous n’aurez peut-être pas besoin d’en passer un autre à la présentation d’une demande de résidence permanente.

À la fin de votre examen, le médecin désigné vous a remis l’un de ces documents :

  • un document d’information; et
  • le formulaire Rapport médical préalable (IMM 1017B).

Lorsque vous faites votre demande en ligne, veuillez fournir une copie de ces documents ou votre numéro d’EMI (ou le numéro d’identificateur unique médical) de votre examen médical précédent.

Si nous ne pouvons pas utiliser vos anciens résultats, nous vous enverrons des instructions sur la façon d’effectuer votre examen médical après avoir présenté votre demande.

3. Payer les frais

Les frais liés à la résidence permanente ont augmenté en avril 2024.

Si vous avez payé vos frais avant le 30 avril 2024, mais que vous n’avez pas encore soumis une demande complète, vous serez tenu de payer la différence.

Le système vous présente la liste des frais que vous devez payer. Cette dernière comprend :

  • vos frais de traitement et les frais de traitement des membres de votre famille;
  • les frais relatifs au droit de résidence permanente;
  • les frais liés à la biométrie.

Découvrez combien vous pouvez vous attendre à payer

Vous devez payer les frais à l’aide de votre compte sécurisé d’IRCC. Vous aurez besoin d’une carte de débit ou de crédit.

Vous devez au minimum payer vos frais de traitement lorsque vous soumettez votre demande. Évitez les retards en payant tous vos frais maintenant (y compris vos frais relatifs au droit de résidence permanente).

Remboursements

Si votre demande n’est pas complète, nous vous en informerons et nous vous rembourserons vos :

  • frais de traitement;
  • frais relatifs au droit de résidence permanente (si vous les avez payés).

Nous ne rembourserons pas les frais de traitement lorsque nous avons entamé le traitement de votre demande.

Payer les frais après avoir fait votre demande

Vous devez payer vos frais qui figurent dans votre compte lorsque vous soumettez votre demande.

Vous pouvez payer ces frais après avoir présenté votre demande, si :

  • vous n’avez pas payé les frais relatifs au droit de résidence permanente après avoir fait votre demande; ou
  • vous souhaitez ajoutez un(e) conjoint(e) ou un enfant à charge à votre demande (mariage ou naissance d’un enfant)

4. Soumettre votre demande

Avant de soumettre votre demande, assurez-vous :

  • d’avoir remplir tous les champs obligatoires du formulaire;
  • de payer les frais de traitement et les frais de biométrie;
  • de fournir des copies numérisées claires de tous les documents figurant sur votre liste de contrôle des documents.

Le système ne vous laissera pas soumettre la demande si vous n’avez pas fourni tous les documents requis et rempli tous les champs obligatoires du formulaire.

Si vous devez mettre à jour des renseignements après avoir soumis une demande

Après avoir soumis une demande, vous pouvez vous servir de ce formulaire Web si vous devez signaler des changements importants concernant votre demande, entre autres :

  • un changement d’adresse, de numéro de téléphone, d’adresse de courriel ou d’autres coordonnées;
  • une naissance ou un décès dans votre famille;
  • un mariage ou un divorce;
  • une adoption;
  • un changement d’emploi ou d’offre d’emploi
  • l’obtention d’un nouveau diplôme;
  • la mise à jour de résultats d’évaluation linguistique.
