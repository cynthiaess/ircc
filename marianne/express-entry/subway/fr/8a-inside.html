Si nous approuvons votre demande : Au Canada

Si nous approuvons votre demande de résidence permanente, vous :

  • devrez confirmer votre statut de résident permanent;
  • recevrez une carte de résident permanent par la poste.

Confirmer votre statut de résident permanent

Dans la plupart des cas, nous pouvons confirmer le statut de résident permanent (RP) :

  • sans entrevue en personne;
  • au moyen d’un portail en ligne sécurisé.

Vous n’avez rien à faire jusqu’à ce que nous vous envoyions un courriel à l’adresse que vous nous avez fournie ou que nous vous appelions au numéro que vous nous avez donné.

Vous recevrez des courriels en provenance d’une adresse se terminant par cic.gc.ca :

  • vous demandant de confirmer votre adresse courriel;
  • vous demandant de confirmer que vous êtes effectivement au Canada;
  • avec des renseignements sur le Portail de résidence permanente (RP) (et les options si vous ne pouvez pas l’utiliser).

Le portail est différent de votre compte sécurisé d’IRCC. Ne créez pas votre propre compte sur le Portail de RP, car cela pourrait retarder le processus. Nous créerons un compte pour vous. Suivez les instructions qui se trouvent dans le courriel que nous vous aurons envoyé.

Comment confirmer votre statut dans le Portail de RP

Vous devez nous informer si vous quittez le Canada avant que nous vous accordions le statut de résident permanent.

Obtenir une carte de résident permanent

Lorsque vous obtiendrez le statut de résident permanent, nous vous fournirons une confirmation de résidence permanente (CRP) électronique dans le Portail de RP afin que vous puissiez présenter une preuve de votre nouveau statut au Canada.

Nous vous demanderons également de nous fournir une photo dans le Portail pour que nous entamions le processus menant à la délivrance de votre première carte de résident permanent (RP). Vous n’avez pas à présenter de demande pour votre première carte RP. Nous enverrons votre carte de RP par la poste à l’adresse canadienne que vous avez indiquée dans votre demande.

En attendant de recevoir votre carte RP, vous pouvez utiliser votre CRP électronique imprimée et signée pour :

  • prouver que vous avez le statut de résident permanent au Canada;
  • demander des avantages et services gouvernementaux auxquels vous avez droit (par exemple, demander un numéro d’assurance sociale pour que vous puissiez travailler).
