Portail pour les établissements d’enseignement désignés: Se connecter

Les établissements d’enseignement postsecondaire utilisent le portail pour les établissements d’enseignement désignés (EED) pour consulter, compléter et soumettre leurs rapports de conformité.

  • Les EED doivent rendre compte du statut de leurs étudiants étrangers deux fois par année (en avril et en novembre).
  • Nous communiquons avec les EED au printemps et à l’automne pour leur rappeler qu’ils doivent soumettre un rapport de conformité.
  • Les EED ont 60 jours pour soumettre leurs rapports de conformité dans le portail.
    • Il n’est pas possible de télécharger les rapports de conformité une fois l’échéance passée.
  • Si vous avez de la difficulté à accéder au portail ou si vous avez des questions au sujet des rapports de conformité, veuillez écrire à l’équipe de l’Autorité pour la gestion de l’intégrité des étudiants.

Le portail pour led EED n'est pas utilisé pour la vérification des lettres d’acceptation.

Erreurs et problèmes lorsque vous ouvrez une session

Vous pouvez utiliser le même nom d’utilisateur et le même mot de passe de CléGC sur tous vos appareils (ordinateurs, téléphones et tablettes).

Problèmes de connexion

Les mises à jour du système peuvent causer des problèmes techniques mineurs. Si vous avez de la difficulté à vous connecter, effacez la mémoire cache de votre navigateur et réessayez.

Si vous effacez l’historique de votre navigateur, la mémoire cache et les fichiers témoins et que vous ne parvenez toujours pas à vous connecter à votre compte sur le Portail des employeurs, essayez de vous connecter en utilisant un autre navigateur (p. ex., Edge, Chrome, Firefox ou Safari).

Si cette solution ne fonctionne pas, envoyez-nous un courriel en y joignant:

  • une description détaillée du problème ou du message d’erreur que vous obtenez;
  • une capture d’écran montrant le problème ou le message d’erreur que vous obtenez;
  • les mesures que vous avez prises pour résoudre le problème; cela nous aidera à mieux comprendre le problème.

Nous vous répondrons dans les dix jours ouvrables.

Si vous obtenez des messages d’erreur :
  • Utilisez la plus récente version d’un navigateur Web pris en charge, comme
    • Safari
    • Chrome
    • Firefox
    • Microsoft Edge

    Si vous utilisez Internet Explorer, mettez à jour votre navigateur en passant à Microsoft Edge.

  • N’ajoutez pas à vos favoris ou à vos signets les pages que vous voyez une fois dans votre compte. Vous devez vous connecter pour les voir.
  • Utilisez les paramètres par défaut de votre navigateur. CléGC nécessite l’activation de JavaScript et des témoins.
  • Si vous voyez l’un de ces messages lorsque vous essayez de vous connecter, mettez votre navigateur à jour :
    • Impossible d’afficher la page
    • Impossible d’établir une connexion sécurisée
    • Page non disponible

    Vous trouverez généralement l’option de mise à jour de votre navigateur dans les paramètres ou les préférences du menu de votre navigateur.

Autres problèmes techniques
  • Trop de tentatives

    Un message d’échec peut s’afficher si vous faites plusieurs tentatives infructueuses pour vous connecter au cours de la même session. Fermez la fenêtre ou l’onglet de votre navigateur et recommencez.

  • Difficultés techniques

    Le site pourrait être indisponible pour diverses raisons, y compris une maintenance du système. Nous nous excusons des inconvénients causés par cette situation. Veuillez réessayer plus tard.

  • Session expirée

    Pour des raisons de sécurité, votre session sera fermée si elle est inactive pendant un certain nombre de minutes. Reconnectez vous.

Vous avez oublié votre nom d’utilisateur ou mot de passe de CléGC

Si vous avez perdu ou oublié votre nom d’utilisateur ou votre mot de passe de CléGC, vous devez nous envoyer un courriel pour récupérer votre compte.

Authentification à deux facteurs de CléGC

L’authentification à deux facteurs protège votre compte. Vous devez choisir une deuxième méthode d’authentification (autre que vos nom d’utilisateur et mot de passe) à utiliser chaque fois que vous vous connectez.

Configurer votre méthode d’authentification à deux facteurs

Modifier votre méthode d’authentification à deux facteurs

Vous devez avoir accès à votre courriel ou application d’authentification existante, ou à vos codes de récupération.

Si vous avez toujours accès à votre méthode à deux facteurs
  1. Connectez‑vous à votre compte avec votre nom d’utilisateur et votre mot de passe de CléGC
  2. Sélectionnez Continuer sur la page d’accueil.
  3. Utilisez votre méthode à deux facteurs actuelle, puis sélectionnez Modifier votre authentification à deux facteurs sur la page de Confirmation d’authentification réussie.
  4. Suivez les invites pour configurer à nouveau l’authentification à deux facteurs.
  5. Conservez vos nouveaux codes de récupération en lieu sûr. Vos anciens codes ne fonctionneront plus.
Si vous n'avez pas accès à votre méthode d'authentification à deux facteurs
Vous avez des codes de récupération :
  1. Connectez‑vous à votre compte avec votre nom d’utilisateur et votre mot de passe de CléGC.
  2. Sur la page d’authentification à deux facteurs, sélectionnez Récupérer le compte.
  3. Entrez l’un des codes de récupération que vous avez enregistrés lors de la première configuration de l’authentification à deux facteurs.
  4. Sur la page de réussite de l’authentification, sélectionnez Modifier votre authentification à 2 facteurs.
  5. Suivez les étapes pour configurer à nouveau l’authentification à deux facteurs.
  6. Conservez vos nouveaux codes de récupération en lieu sûr. Vos anciens codes ne fonctionneront plus.
Vous n’avez pas de codes de récupération :

Vous devez nous envoyer un courriel pour récupérer votre compte. Vous devrez ensuite configurer de nouveau l’authentification à deux facteurs.

Communiquer avec le Bureau de service de CléGC

Les représentants de CléGC peuvent vous aider uniquement pour des problèmes concernant la CléGC (y compris l’authentification à deux facteurs). Ils n’ont pas accès aux renseignements sur les demandes et les comptes du gouvernement du Canada.

Les représentants du service CléGC sont disponibles pour vous aider par téléphone toute l’année, 24 heures sur 24, 7 jours sur 7.

Obtenir plus d’aide sur l’authentification à deux facteurs.

CléGC révoquée

Si votre nom d’utilisateur and votre mot de passe de CléGC ont été révoqués, vous ne pourrez pas les récupérer. Ils peuvent être révoqués si :

  • votre nom d’utilisateur et votre mot de passe ont été compromis;
    • Le nom d’utilisateur et le mot de passe que vous utilisez pour CléGC doivent être différents de ceux associés à vos autres comptes en ligne.
  • vous n’avez pas utilisé CléGC depuis deux ans.

Si votre nom d’utilisateur et votre mot de passe de CléGC ont été révoqués, vous devez nous envoyer un courriel pour récupérer votre compte. Vous pouvez choisir une autre méthode de connexion ou refaire une demande de nom d’utilisateur et de mot de passe de CléGC

Changer votre partenaire de connexion

Si vous avez changé d’institution bancaire ou de coopérative de crédit, il est possible que vous puissiez changer de partenaire de connexion :

  1. sélectionnez Partenaire de connexion canadien Interac®;
  2. sélectionnez Changer mon partenaire de connexion dans le menu de la page Sélectionner un partenaire de connexion;
  3. suivez les étapes pour changer votre partenaire de connexion.
