Présenter une demande de visa de visiteur en ligne : portail d’IRCC

Suivez ces étapes pour présenter votre demande en ligne à l’aide du portail d’IRCC.

1. Obtenir un code d'invitation, créer un compte et ouvrir une session

Si vous utilisez le portail d’IRCC pour la première fois, vous avez besoin d’un code d’invitation pour créer un compte. Une fois que vous avez votre code, vous pouvez créer un compte sur le portail et ouvrir une session.

2. Remplir le formulaire en ligne et téléverser vos documents

Après avoir répondu à toutes les questions, vous obtiendrez une liste des documents que vous devez téléverser.

Ceci inclut les documents appuyant votre demande de visa de visiteur.

3. Payer avec une carte de crédit

Nous acceptons les cartes de crédit et les cartes prépayées de VisaMD, MasterCardMD, American ExpressMD, JCBMD, et UnionPayMD.

Si vous ne pouvez pas présenter votre demande en ligne

Vous pouvez faire une demande sur papier si vous ne pouvez pas faire de demande en ligne :

  • en raison d’un handicap;
  • parce que vous voyagez avec un passeport pour étranger apatride ou un autre titre de voyage pour réfugiés non-citoyens.
