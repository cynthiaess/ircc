Vous ne savez pas trop quel compte vous avez utilisé?

Si vous avez ouvert une session à l’aide d’un nom d’utilisateur et d’un mot de passe CléGC, ou avec votre partenaire de connexion bancaire canadien, vous avez utilisé le compte sécurisé d’IRCC.

Si vous avez ouvert une session à l’aide d’une adresse courriel, vous avez utilisé le portail d’IRCC.

Si vous ne savez toujours pas quel compte vous avez utilisé, essayez de vous connecter à chaque compte ou vérifiez votre courrier électronique.

Vous ne savez pas trop quel compte vous avez utilisé?

Vous avez utilisé le compte sécurisé d’IRCC si :

  • vous avez ouvert une session à l’aide d’un nom d’utilisateur et d’un mot de passe CléGC ou
  • vous avez ouvert une session à l’aide de votre partenaire de connexion bancaire canadien

Vous avez utilisé le portail d’IRCC si :

  • vous avez ouvert une session à l’aide d’une adresse courriel et
  • vous avez présenté une demande pour obtenir un visa de visiteur ou un permis d’études

Vous avez utilisé le portail de résidence permanente si :

  • vous avez ouvert une session à l’aide d’une adresse courriel et
  • vous avez présenté une demande pour obtenir la résidence permanente, une carte de résident permanent (RP) ou un titre de voyage pour résident permanent (TVRP)

Si vous ne savez toujours pas quel compte vous avez utilisé, essayez de vous connecter à chacun d’eux ou vérifiez votre courrier électronique.
