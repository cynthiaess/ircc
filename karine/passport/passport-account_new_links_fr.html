Renouvelez votre passeport en ligne

Vous pourrez renouveler votre passeport en ligne si vous remplissez certaines conditions.

Qui peut renouveler son passeport en ligne?

Vous devez remplir toutes les conditions suivantes pour pouvoir renouveler votre passeport en ligne :

  •  Vous présentez une demande de renouvellement de votre passeport.
  •  Votre domicile et votre adresse postale se trouvent au Canada.
  •  Votre passeport actuel :
    • est un passeport ordinaire (bleu)
    • était valide pour 5 ou 10 ans
    • indique votre lieu de naissance.
    • a été délivré lorsque vous aviez 16 ans ou plus
    • a été délivré au cours des 15 dernières années et
      • a expiré ou
      • expire au cours des six prochains mois
  •  Vous utiliserez le même nom, la même date de naissance, le même lieu de naissance et la même désignation de genre sur votre nouveau passeport.
  •  Vous n’avez aucune observation dans votre passeport actuel.
    Qu’est-ce qu’une observation?

    Les observations sont des notes dans une page séparée du passeport de certaines personnes pour expliquer des circonstances particulières (par exemple, un nom trop long pour être imprimé au complet, ou s’il s’agit d’un passeport diplomatique). La plupart des gens n’ont pas d’observations dans leur passeport.

  •  Vous n’avez pas déclaré que votre passeport actuel avait été saisi ou remis.
  •  Vous n’avez pas besoin de votre nouveau passeport au cours des 20 jours ouvrables (plus le délai de livraison par la poste).
  •  Vous avez une photo de passeport numérique.
    • Lisez et imprimez les spécifications pour les photos de passeport numériques.

Créez un compte maintenant, même si vous n'êtes pas prêt à renouveler votre passeport

Nous offrons de manière aléatoire la possibilité de renouveler votre passeport en ligne. Si vous quittez cette page, il se peut que cette option ne vous soit plus proposée. Si vous remplissez les conditions, créez un compte afin d'être prêt à renouveler votre passeport en ligne à tout moment.

Aide avec votre compte

Erreurs et problèmes lorsque vous ouvrez une session

Vous pouvez utiliser le même nom d’utilisateur et le même mot de passe de CléGC sur tous vos appareils (ordinateurs, téléphones et tablettes).

Si vous obtenez des messages d’erreur :
  • Utilisez la plus récente version d’un navigateur Web pris en charge, comme
    • Safari;
    • Chrome;
    • Firefox;
    • Microsoft Edge;

    Si vous utilisez Internet Explorer, mettez à jour votre navigateur en passant à Microsoft Edge.

  • N’ajoutez pas à vos favoris ou à vos signets les pages que vous voyez une fois dans votre compte. Vous devez vous connecter pour les voir.
  • Utilisez les paramètres par défaut de votre navigateur. CléGC nécessite l’activation de JavaScript et des témoins.
  • Si vous voyez l’un de ces messages lorsque vous essayez de vous connecter, mettez votre navigateur à jour :
    • Impossible d’afficher la page
    • Impossible d’établir une connexion sécurisée
    • Page non disponible

    Vous trouverez généralement l’option de mise à jour de votre navigateur dans les paramètres ou les préférences du menu de votre navigateur.

Autres problèmes techniques
  • Trop de tentatives

    Un message d’échec peut s’afficher si vous faites plusieurs tentatives infructueuses pour vous connecter au cours de la même session. Fermez la fenêtre ou l’onglet de votre navigateur et recommencez.

  • Difficultés techniques

    Le site pourrait être indisponible pour diverses raisons, y compris une maintenance du système. Nous nous excusons des inconvénients causés par cette situation. Veuillez réessayer plus tard.

  • Session expirée

    Pour des raisons de sécurité, votre session sera fermée si elle est inactive pendant un certain nombre de minutes. Reconnectez vous.

Vous n'avez pas de code de vérification de compte

Vous devez créer un compte avant de vous connecter, même si vous disposez déjà d'un nom d'utilisateur et d'un mot de passe CléGC ou de renseignements bancaires. Lors de votre inscription, vous recevrez un code de vérification à usage unique que vous utiliserez lorsque vous vous connecterez à votre compte pour la première fois.

Vous ne savez pas quel code de vérification utiliser
Vous vous inscrivez en utilisant un nom d'utilisateur et un mot de passe CléGC
Vous avez déjà un nom d'utilisateur et un mot de passe CléGC

Lorsque vous ouvrirez un compte sur le Portail IRCC – Nouvelle version, vous recevrez un code de vérification unique dont vous aurez besoin lors de votre première connexion. Ce code vous a été envoyé à l'adresse courriel avec laquelle vous vous êtes inscrit.

Vous créez un nouveau nom d'utilisateur et un nouveau mot de passe CléGC

Lorsque vous créerez un compte sur le Portail IRCC – Nouvelle version, vous recevrez un code de vérification à usage unique dont vous aurez besoin lors de votre première connexion. Mais avant cela, vous allez créer un nom d'utilisateur et un mot de passe CléGC et configurer votre authentification à deux facteurs.

Vous recevrez également un code de vérification lorsque vous configurerez votre CléGC et un autre lorsque vous configurerez votre authentification à deux facteurs. Vous utiliserez ces codes avant d’utiliser le code de vérification du compte que vous avez déjà reçu. Une fois que vous aurez établi votre nom d'utilisateur et votre mot de passe CléGC et complété votre authentification à deux facteurs, vous pourrez vous connecter à votre compte du Portail IRCC – Nouvelle version et utiliser le premier code de vérification que vous avez reçu.

Vous vous inscrivez en utilisant le partenaire de connexion

Lorsque vous créerez un compte sur le Portail IRCC – Nouvelle version, vous recevrez un code de vérification unique dont vous aurez besoin lors de votre première connexion. Ce code vous sera envoyé à l'adresse courriel avec laquelle vous vous êtes inscrit.

Vous avez oublié votre nom d’utilisateur ou mot de passe de CléGC

Veuillez lire les instructions correspondant à votre situation.

Si vous avez oublié votre nom d’utilisateur et votre mot de passe
  1. Suivez d’abord les étapes pour récupérer votre nom d’utilisateur.
  2. Une fois que vous avez récupéré votre nom d’utilisateur, suivez les étapes pour récupérer votre mot de passe.
Vous avez perdu ou n’utilisez plus l’adresse courriel que vous avez utilisée pour vous inscrire

Communiquez avec nous en utilisant notre formulaire Web pour savoir ce que vous devez faire.

Vous avez oublié votre nom d’utilisateur
Si vous avez une adresse courriel de récupération :
  1. sélectionnez Nom d’utilisateur et mot de passe de CléGC;
  2. sélectionnez Vous avez oublié votre nom d’utilisateur? sur la page Bienvenue au service CléGC;
  3. saisissez votre adresse courriel de récupération sur la page Récupérer le nom d’utilisateur – Étape 1 sur 2;
    • vous avez fourni votre courriel de récupération lorsque vous vous êtes inscrit à CléGC, ou à un moment ultérieur;
  4. sélectionnez Continuer;
    • vous serez dirigé vers une page qui explique à quelle adresse courriel nous enverrons votre nom d’utilisateur et ce qu’il faut faire si vous ne recevez pas le courriel;
  5. sélectionnez Continuer;
  6. vérifiez vos courriels;
    • vous devriez recevoir un courriel du gouvernement du Canada contenant votre nom d’utilisateur de CléGC;
  7. connectez vous avec votre nom d’utilisateur et votre mot de passe sur la page Bienvenue au service CléGC.
Si vous n’avez pas d’adresse courriel de récupération :

Communiquez avec nous en utilisant notre formulaire Web pour savoir ce que vous devez faire.

Vous avez oublié votre mot de passe
  1. Sélectionnez Nom d’utilisateur et mot de passe de CléGC.
  2. Sélectionnez Vous avez oublié votre mot de passe? sur la page Bienvenue au service CléGC.
  3. Saisissez votre nom d’utilisateur de CléGC.
  4. Sélectionnez Continuer.
    • Si vous avez saisi une adresse courriel pour obtenir votre nom d’utilisateur de CléGC, réinitialisez votre mot de passe avec votre adresse courriel.
    • Si vous n’avez pas saisi d’adresse courriel lors de votre inscription, réinitialisez votre mot de passe à l’aide des questions de récupération.
Réinitialisation du mot de passe à l’aide de l’adresse courriel
  1. Saisissez votre adresse courriel.
  2. Vérifiez vos courriels. Vous devriez recevoir un courriel du gouvernement du Canada contenant un code de confirmation.
  3. Saisissez le code de confirmation sur la page Vous avez oublié votre mot de passe? - Étape 2 sur 4.
    • Cette page devrait s’être chargée automatiquement après que vous avez saisi votre nom d’utilisateur et sélectionné Continuer sur la page précédente.
  4. Sélectionnez Continuer.
  5. Vous serez invité à répondre à une question de récupération. Saisissez la réponse à la question dans le champ Ma réponse de récupération.
  6. Sélectionnez Continuer.
  7. Saisissez un nouveau mot de passe dans le champ Nouveau mot de passe.
  8. Saisissez de nouveau le mot de passe dans le champ Confirmer le nouveau mot de passe.
  9. Sélectionnez Continuer.
  10. Sélectionnez encore une fois Continuer sur la page Vous avez oublié votre mot de passe – Complété.
Réinitialisation du mot de passe à l’aide des questions de récupération
  1. Sur la page Vous avez oublié votre mot de passe? – Étape 2 sur 4, sélectionnez Questions et réponses de récupération dans la case Questions et réponses de récupération.
  2. Saisissez vos réponses dans les champs Ma réponse de récupération, Ma personne mémorable et Ma date mémorable.
    • Les questions et indices devraient s’afficher au dessus de chaque champ.
  3. Saisissez un nouveau mot de passe dans le champ Créez votre nouveau mot de passe.
  4. Saisissez le même mot de passe dans le champ Confirmez votre nouveau mot de passe.
  5. Sélectionnez Continuer.
  6. Sélectionnez encore une fois Continuer sur la page Vous avez oublié votre mot de passe? - Complété
    • Si vous avez répondu correctement à toutes les questions de récupération, vous serez automatiquement dirigé vers cette page.

Authentification à deux facteurs de CléGC

L’authentification à deux facteurs protège votre compte. Vous devez choisir une deuxième méthode d’authentification (autre que vos nom d’utilisateur et mot de passe) à utiliser chaque fois que vous vous connectez.

Configurer votre méthode d’authentification à deux facteurs

Modifier votre méthode d’authentification à deux facteurs

Vous devez avoir accès à votre courriel ou application d’authentification existante, ou à vos codes de récupération.

Si vous avez toujours accès à votre méthode à deux facteurs
  1. Connectez vous à votre compte avec votre nom d’utilisateur et votre mot de passe de CléGC
  2. Sélectionnez Continuer sur la page d’accueil.
  3. Utilisez votre méthode à deux facteurs actuelle.
  4. Sur la page de Confirmation d’authentification réussie, sélectionnez Modifier votre authentification à deux facteurs.
  5. Sur la page de d’authentification à deux facteurs, suivez les étapes pour configurer à nouveau l’authentification à deux facteurs.
  6. Conservez vos nouveaux codes de récupération en lieu sûr. Vos anciens codes ne fonctionneront plus.
Si vous n'avez pas accès à votre méthode d'authentification à deux facteurs
Vous avez des codes de récupération :
  1. Connectez vous à votre compte avec votre nom d’utilisateur et votre mot de passe de CléGC.
  2. Sur la page d’authentification à deux facteurs, sélectionnez Récupérer le compte.
  3. Entrez l’un des codes de récupération que vous avez enregistrés lors de la première configuration de l’authentification à deux facteurs.
  4. Sur la page de réussite de l’authentification, sélectionnez Modifier votre authentification à 2 facteurs.
  5. Suivez les étapes pour configurer à nouveau l’authentification à deux facteurs.
  6. Conservez vos nouveaux codes de récupération en lieu sûr. Vos anciens codes ne fonctionneront plus.
Vous n’avez pas de codes de récupération :

Communiquez avec nous en utilisant notre formulaire Web pour savoir ce que vous devez faire.

Communiquer avec le Bureau de service de CléGC

Les représentants de CléGC peuvent vous aider uniquement pour des problèmes concernant la CléGC (y compris l’authentification à deux facteurs). Ils n’ont pas accès aux renseignements sur les demandes et les comptes du gouvernement du Canada.

  • Au Canada et aux États Unis : 1-855-438-1102
  • Téléphone textuel (TTY/ATS) : 1-855-438-1103
  • À l’extérieur du Canada et des États Unis : 1-800-2318-6290

Les représentants du service CléGC sont disponibles pour vous aider par téléphone toute l’année, 24 heures sur 24, 7 jours sur 7.

Obtenir plus d’aide sur l’authentification à deux facteurs.

CléGC révoquée

Si votre nom d’utilisateur and votre mot de passe de CléGC ont été révoqués, vous ne pourrez pas les récupérer. Ils peuvent être révoqués si :

  • votre nom d’utilisateur et votre mot de passe ont été compromis;
  • vous n’avez pas utilisé CléGC depuis deux ans.
    • Le nom d’utilisateur et le mot de passe que vous utilisez pour CléGC doivent être différents de ceux associés à vos autres comptes en ligne.

Si votre nom d’utilisateur et votre mot de passe CléGC ont été annulés, communiquez avec nous en remplissant notre formulaire Web pour savoir ce que vous devez faire.

Changer votre partenaire de connexion

Si vous avez changé d’institution bancaire ou de coopérative de crédit, il est possible que vous puissiez changer de partenaire de connexion :

  1. sélectionnez Partenaire de connexion canadien Interac®;
  2. sélectionnez Changer mon partenaire de connexion dans le menu de la page Sélectionner un partenaire de connexion;
  3. suivez les étapes pour changer votre partenaire de connexion.
